# 状態遷移の仕様

## 状態の種類

- 審査中
- 承認済
- 差戻中
- 実施中
- 中断中
- 終了

## 状態遷移図

| **from/to** | 審査中 | 承認済 | 差戻中   | 実施中 | 中断中 | 終了     |
| ----------- | ------ | ------ | -------- | ------ | ------ | -------- |
| 審査中      | -      | 承認   | 差し戻し | -      | -      | -        |
| 承認済      | -      | -      | -        | 開始   | -      | 取り下げ |
| 指戻中      | 再申請 | -      | -        | -      | -      | 取り下げ |
| 実施中      | -      | -      | -        | -      | 中断   | 完了     |
| 中断中      | -      | -      | -        | 終了   | -      | 中止     |
| 終了        | -      | -      | -        | -      | -      | -        |
